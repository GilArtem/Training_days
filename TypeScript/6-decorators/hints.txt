Декораторы являются инструментом декларативного программирования, они позволяют добавить
к классам и их членам метаданные и тем самым изменить их поведение без изменения их кода.

Декораторы представляют функции, которые могут применяться к классам, методам, методом доступа 
(геттерам и сеттерам), свойствам, параметрам.

При компиляции следует указывать параметр experimentalDecorators.
Например, через файл tsconfig.json:
`
{
    "compilerOptions": {
        "target": "ES5",
        "experimentalDecorators": true
    }
}
`
